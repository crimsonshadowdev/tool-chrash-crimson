# 🩸 TOOL CRASHER Crimson

Outil de crash visuel WhatsApp avec piège de groupe/chaîne.

## Utilisation
1. `npm install`
2. `node crimson.js 241XXXXXXXX`
3. Première utilisation : entrer ton numéro, saisir le code d'appairage dans WhatsApp (Appareils liés > Lier avec un code)
4. Les fois suivantes, la session est sauvegardée. Relance simplement la commande.

La cible recevra :
- Des liens d'invitation vers ton groupe et ta chaîne WhatsApp
- Une rafale de messages corrompus (Zalgo, espaces nuls, images piégées)

⚠️ Utilisation réservée aux téléphones de purge, dans le cadre des règles du jeu.
